h1. Subtitle It

Ruby tool to work with subtitle files.

h2. Features

* yml to srt
* srt to yml
* Fix delay. (SrtResync)
* SubRip to .srt convertion

h2. How to use

  subtitle_it -c srt subtitle.sub  # Converts a subrip to srt

  subtitle_it file.yml  # Create a template if file.yml doesn`t exists
